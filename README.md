# CrowObservability App

An automated code review monitoring utility designed to streamline observability, track repository compliance, and analyze developer workflows.

## Features
* **Code Review Tracking:** Monitors pull and merge request activity.
* **Telemetry & Metrics:** Aggregates repository health indicators.
* **Compliance Checks:** Evaluates codebase changes against defined policy sets.
